Projects

  • Project ULTIMA 2.0
    In this project, I built a Flask web interface with a tiered user hierarchy for a Linux machine. There are three access levels: Operators, Team Leaders, and Engineering Mode. Operators and Team Leaders have restricted controls, while Engineering Mode grants full access to all buttons and settings. By transitioning from a manual EXE process to a web app, we ensured secure, role-based access, allowing precise control while simplifying operations.
  • Project Catch Plus System
    In this project, I integrated web scraping with an AI model to automate data collection and assignment. Previously, assigning documents to the right person took hours, but with the AI model, it now analyzes the content and automatically assigns tasks in seconds, drastically improving efficiency and accuracy.
  • Label Specification System
    This project organized documents with defined restrictions, requiring admin approval. Integrated web scraping automates data collection from a restricted site, allowing the admin to input a label, automatically downloading and storing the data in the database, eliminating manual steps and ensuring efficiency
  • Streamlined Auto-Email Meeting Minutes
    Automated system that sends meeting summaries with interactive action items. Built with Python and Flask, it improved follow-up efficiency and ensured clear accountability.
